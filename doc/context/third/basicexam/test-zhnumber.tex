\section{零的设置}

默认使用〇：
\startitemize[n,horizontal,columns][n=2]
\item 101: \zhnumber{101}
\item 2024年: \zhdate{2024/1/1}
\stopitemize

\setupzhnumber[zero=零]

切换为零：
\startitemize[n,horizontal,columns][n=2]
\item 101: \zhnumber{101}
\item 2024年: \zhdate{2024/1/1}
\stopitemize

\setupzhnumber[zero=〇]

切换回〇：
\startitemize[n,horizontal,columns][n=2]
\item 101: \zhnumber{101}
\item 2024年: \zhdate{2024/1/1}
\stopitemize
