\title{综合排序测试}

\section{排序方法说明}

本测试文档演示了三种中文排序方法：

\startitemize
\item \bf{zh-pinyin}：按拼音排序，同音字按拼音字母顺序排列
\item \bf{zh-alpha}：按字母顺序排序，汉字按拼音字母顺序排列
\item \bf{zh-stroke}：按笔画数排序，同笔画数按笔顺代码排列
\stopitemize
